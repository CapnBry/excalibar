ZLib Version 1.1.3

This is the newest ZLib-Version 1.1.3 for use with Delphi, 
compiled with the Borland C++ Compiler 5.5.

It is meant as replacement for the older version 1.0.4, 
which is delivered with Delphi 3-5. 

All sources come from the ZLib homepage: 
http://www.info zip.org/pub/infozip/zlib.
